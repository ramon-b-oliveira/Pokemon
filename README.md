# Pokémon Guessing Game

A browser-based Pokémon guessing game built with Vue 3 and TypeScript. The app fetches Pokémon data from the PokéAPI and challenges players to identify Pokémon from their silhouettes.

## Tech Stack

- **Vue 3** with Composition API
- **TypeScript** for type safety
- **PokéAPI** for real-time Pokémon data
- **Font Awesome** for icons
- **CSS** custom styling and animations

## How to Run

```bash
# Clone the repo
git clone https://github.com/ramon-b-oliveira/Pokemon.git
cd Pokemon/pokeguess

# Install dependencies
npm install

# Start dev server
npm run dev
```

## Features

- Pokémon silhouette guessing gameplay
- Real-time data from PokéAPI
- Score tracking and feedback
- Smooth transitions and animations
